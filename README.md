# TravelTrucks

TravelTrucks is a React-based web application for browsing, filtering, and booking campervans.  
The project follows a provided Figma design and uses a public mock API for camper data.

---

## Tech Stack

- **React**
- **Vite**
- **Redux Toolkit**
- **React Router**
- **CSS Modules**
- **Axios**

---

## Features

- Home page with hero section
- Catalog page with:
  - Camper cards
  - Filters (location, equipment, vehicle type)
  - Load more pagination
- Camper details page with:
  - Features
  - Reviews
  - Booking form
- Favorites (stored in `localStorage`)
- UI aligned with Figma design

---

## Setup & Run

```bash
npm install
npm run dev
```

Open in browser:
http://localhost:5173
